DESCARGA XML (ADD)

Este módulo te permite descargar de la página del SAT fácil y rápidamente tus XML emitidos y recibidos

Descarga tus facturas del SAT de manera automática y sencilla!!

  • El módulo depende de las librerías xmltodict y openssl, para instalar en Ubuntu utiliza los siguientes comandos:
    sudo pip3 install xmltodict
    sudo apt-get install python3-openssl


    O bien, contactanos para brindarte apoyo en la instalacion del módulo.

  • INSTRUCCIONES DE USO:
  • 1. Agrega tu FIEL para descargar las facturas en Configuración -> Compañia -> CFDI 4.0 Sat Sync




    • 2. Selecciona la última fecha desde la que quieres que se descarguen tus facturas


    • 3. Visualiza tus documentos descargados


    • 4. Selecciona una de las opciones para crearlos localmente
